### 富有的习惯承诺2：目标设定与全心投入

**我会设定每日、每月、每年的目标以及长期目标，然后全心投入追求自己的目标。**

---

## 成功人士的行为与目标设定

### 目标导向的行为
- **设定目标**：成功人士总是为自己设定目标，并将日常目标写进待办事项中。
- **专注工作**：他们工作时心无旁骛，暂时抛开个人和家庭事务。
- **长期规划**：设定长期目标，确保在特定时间点前完成重要任务。

### 长期视角
- **放眼未来**：成功人士从长期目标出发，确定自己在完成目标过程中的位置。
- **不断调整**：他们不纠缠于过去的成功或失败，而是不断调整方向，确保每一步都朝着目标迈进。

### 非成功人士的行为
- **缺乏目标**：非成功人士往往没有明确的目标，容易分心，难以获得成功。

---

## 当日目标示例

| 描述                                | 完成时间       |
|-------------------------------------|----------------|
| 阅读与自身行业有关的资料            | 6:00—6:30      |
| 检查语音信箱和电子邮件              | 8:30           |
| 拨出一个问候电话                    | 9:00           |
| 拨出或发出生日祝福                  | 9:00           |
| 给潜在客户打电话                    | 9:30           |
| 完成尚未确定的人寿保险申请单        | 10:00          |
| 跟进尚未确定的案例                  | 11:00          |
| 与客户会面                          | 13:00—17:00    |
| 回复所有的电话                      | 17:00          |
| 再次给潜在客户打电话                | 17:30          |

**备注**：每天完成80%的当日目标即为成功。

---

### 富有的习惯：当日目标/待办事项清单
- **完成率**：每天完成80%的当日目标即为成功。
- **时间管理**：为每个任务设定具体的完成时间。
- **优先级**：为低优先级的任务设定较低的优先权，留待第二天完成。

## 年度与长期目标示例

| 今年目标                            | 明年目标                            | 长期目标                            |
|-------------------------------------|-------------------------------------|-------------------------------------|
| 参加注册会计师考试                  | 写书并出版                          | 5年内挣够……美元                    |
| 获得长期照护资格                    | 另存……美元                          | 给妻子买一辆敞篷车                  |
| 存款增加……美元                      | 参加注册理财规划师考试              | 到……之前，成为合伙人                |
| 还清汽车贷款                        | 体重减到80公斤                      | 到……之前付清房贷                    |
| 到……之前体重减轻4.5公斤             | 跑马拉松                            | 到……之前买海滨别墅                  |
| 升到……的职位                        | 薪水增加到……美元                    | 给孩子的大学教育存款增加到……美元    |

**备注**：设定明确的目标并拆分为具体任务，有助于实现长期愿景。

---

## 愿景板
使用愿景板可以帮助保持长期目标的可见性。通过将长期目标转化为实实在在的画面，如梦想中的房子照片、公司照片等，可以更好地激励自己。

### 示例
- **金·凯瑞**：在无名时期曾写下一张2000万美元的支票，并将其放在每天都能看到的地方。这张支票代表他希望达到的长期目标——出演一部电影，拿到2000万美元的报酬。最终，他如愿以偿。

---

## 小结
成功人士会设定目标，并为达到这些目标拟定具体的计划。通过制定当日、当月、今年和来年的目标，以及长期目标，他们能够更好地管理时间和资源，最终实现成功。
