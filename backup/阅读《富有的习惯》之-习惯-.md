### 富有的习惯承诺1：培养自己的日常富有的习惯，每天照做不误

良好的日常习惯是成功的基石。成功人士与普通人的区别在于他们的日常习惯。他们养成了许多好习惯，几乎没有坏习惯，而不成功的人则相反，有许多坏习惯，好习惯却屈指可数。大部分成功人士甚至意识不到自己拥有这么多的好习惯，因此定义成功并不容易。

如果不能清楚地认识自己的强项和弱项，任何人都无法取得成功。我们必须对自己诚实，不留情面，才能做出准确的评估。然而，什么是强项，什么是弱项？由于标准模糊，很难给出明确的定义。此外，自尊心可能会干扰我们的判断，导致分析结果不准确。因此，依赖第三方进行评估可能比自我评估更为容易。

有一个简单可靠的方法可以帮助我们进行自我评估，那就是检视每天的生活方式与日常习惯。通过了解自己的日常习惯，我们能够对自己的强项和弱项获得准确的洞察。糟糕的日常习惯会阻碍我们走向成功，认识到这一点，是迈向财务成功的第一步，也是最关键的一步。

#### 自我评估表

在一张纸上画出两列表格。在第一列中填上自己糟糕的日常习惯，在第二列中填上对应的好习惯。第二列将成为我们培养好习惯的清单。例如：

| 坏习惯                     | 好习惯                                   |
|----------------------------|------------------------------------------|
| 我每天看电视的时间太多。     | 我限制自己每天只看1小时电视。             |
| 我没有定期锻炼。             | 我每天锻炼30分钟。                       |
| 我从不关心自己吃了什么。     | 我每天摄入的热量不能超过××卡路里。       |
| 我从不阅读与自己专业有关的书籍。 | 我每天阅读30分钟。                       |
| 我做事拖拉。                 | 我每天都要把待办日程上的工作做完。       |
| 我抽烟。                     | 今天我不抽烟。                           |
| 我花了太多时间上网。         | 我限制每天用来刷网页的时间。             |
| 我不马上回电话。             | 今天我要及时回复每一个电话。             |
| 我记不住别人的名字。         | 我把人名写下来，然后背熟。               |
| 我总是忘记重要的约会。       | 我了解说定的约会对别人来说很重要。       |

> [!TIP]
> 成功人士在生活中严格遵循良好的日常习惯，这是第一条，也是最重要的一条富有的习惯。

通过这种方式，我们可以逐步培养出富有的习惯，迈向成功。
